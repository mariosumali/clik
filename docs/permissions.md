# Permissions Matrix

## Required Permissions

- Accessibility: AX targeting and protected UI interactions.
- Input Monitoring: global hotkeys and macro recording event taps.
- Screen Recording: image/color/OCR targeting and preview capture.
- Microphone: audio-based triggers.

## Degraded Behavior

- Missing permissions only disable the related feature.
- App remains functional for non-dependent features.
- Settings includes one-click deep links into macOS privacy panes.
